reMember

This is a very simplistic tool for converting hand-written notes (e.g., on a reMarkable) to Anki flashcards. It works by detecting lines surrounding flashcards as in the example below and crops the image appropriately. At the moment this is extremely simplistic and all configuration options are hidden in the code, but I plan on expanding it in the future.

Note: Everything runs in your browser and no data is sent anywhere!

Just drop an image file below and the flashcards will be generated, triggering a download of an Anki package.

Input

Leave empty to generate a random deck ID. The deck ID is also used to compute card IDs by numbering them from top to bottom.
